Esta web esta destinada a ser un registro monetario en el cual cada persona podrá ir anotando todos los gastos que tiene en el día a día. Para así a final de mes tener un resumen de gastos dividido en categorias.

El usuario objetivo sería cualquier persona que quiera tener un seguimiento de sus gastos diariamente para así poder saber donde acaba su dinero, y si lo está aprovechando de una manera eficiente.

# Funcionalidades principales
1. Registro de gastos expecificando: cantidad, categorías, fecha y descripción opcional.
2. categorías como: comida, transporte, ocio, ropa, salud, otros.
3. Resumen mensual detallado.
4. Guardar los datos en LocalStorage para que no se borren los datos al cerrar la pagina.

# Funcionalidades opcionales
1. filtro de gastos por categorías
2. Realizar presupuesto por categoría y avisarte de alguna forma si lo superas

# Mejoras futuras
1. Representación gráfica de los gastos por categoría
2. Comparar gastos globales entre meses
3. Adjuntar ingresos para hacer un balance real
