# Expense Tracker

Aplicación fullstack para registrar gastos personales y ver resúmenes mensuales por categoría.

## Gestión del proyecto

- **Tablero Trello:** [Ver tablero](https://trello.com/b/8mYPpA80/expense-tracker)

## Stack

- React 18 + TypeScript + Vite
- Tailwind CSS
- React Router
- Node.js + Express (próximamente)

## Estructura del proyecto

src/
├── components/   # Componentes reutilizables
├── pages/        # Páginas de la app
├── hooks/        # Custom hooks
├── types/        # Interfaces y tipos TypeScript
├── utils/        # Funciones auxiliares
├── context/      # Context API
└── api/          # Cliente de API
